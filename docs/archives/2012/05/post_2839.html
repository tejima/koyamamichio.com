パソコン修理

快晴　真夏の猛暑


午前７時朝食　ブンボー・フエ（フエ名物牛肉入り米麺）



パソコンは中国からの「ウイスル」に侵され、使用不能に。
インターネットのどこを開いても中国の文章や写真が出てくる。

ソン君と専門業者が来て、必要な部分の修理。
３０％ほど使えるようになったが、基本的には、全面修理が
必要。

午前中は、パソコンの状況を見てもらう。

明日、午前中に「現在のウインドウズ」を削除し、「XP」を
入れることにした。

そのため、午前中と午後、パソコンに履いている様々な資料などを
USBに移す作業を行う。

相当時間がかかる。


午前１１時半昼食。

１時間ほど昼寝。

午後は、通販「フェリシモ」から頂いた寄金を使って
「子どもの家」の改修などを行った。
活用の報告書と必要な写真１０枚ほどを用意する。

１　「子どもの家」全体の電気の配線の総合修理
２　「子どもの家」全体の水道管の全面改修・整備
３　「子どもの家」の遊具の改修
４　「子どもの家」音楽室のピアノの調律と修繕
５　自習室の机の新調
６　「子どもの家」の割れたガラスの改修
７　こどもたちの語学学習用のCD等の購入
８　「子どもの家」の成長を記録し、こどもたち「子どもの家」を
　　卒業する際にお祝いに贈る「アルバム」用の写真を取るため
　　のデジカメ購入


夜、「フェリシモ」に寄金の活動報告書と写真を送信。
