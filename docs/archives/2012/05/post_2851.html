2012年05月27日

バンベー４１号編集：各種文書作成

快晴　　猛暑


この３日間、ブライセン本社の４人を迎えての支社研修で
相当疲れた。
今朝は、起きられなかった。半日寝ていた。

昼、センさんが昨日作っておいてくれた「マカロニスープ」を
半分ほど食べる。食欲なし。

午後も横になっていた。


ーーー

５月２５日（金）は「子どもの家」出身であるソン君（ベトナム事務所員）
とメンさん（フエ市人民委員会外務部員）の長男「バオ君」の
１歳の誕生日だった。おめでとう。健康に成長していって欲しい。


投稿者 koyama : 2012年05月27日 22:28
