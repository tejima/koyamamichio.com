午前１１時半、昼食。チャオ・バイン・カイン（小麦粉の手打ちうどん）



体がだるい。２時間ほど昼寝。
室内も猛暑。いるだけで汗が出てくる。エヤコンは、効かない。
それほど、太陽の日差しが強い。

午後３時から１時間、ウオーキング。
強い日差しの中を歩く。出来るだけ木陰の下を。
