2014年07月22日

ドンバ市場値段調査：日本料理店

終日猛暑

午前６時起床。

午前７時朝食

「フォー・ボー」（牛肉米麺）


午前９時、ベトナム事務所へ。

内田さんがベトナム事務所でバオミンさんと懇談。


私はハンさんと一緒にドンバ市場へ。

函館の会からの要請で、函館のボランティア会
に出展する為に「チョコレート、スナック菓子・・・」の
値段を調査して欲しいとの依頼を受けて、
ドンバ市場へ行く。

乾燥野菜（５００グラム　２２５円）

乾燥バナナ　４００グラム２２０円

午前１１時半昼食。

２時間程昼寝。


２０周年式典の挨拶文を仕上げる。

ベトナム人の挨拶をバオミンさんがベトナム語から
日本語に直したものを、再度日本語に直す。


午後５時半、徒歩で日本料理店へ。


川原先生・富岡先生が日本語学校から帰る。


開店から閉店まで内田さんが日本料理店の応援に


今日は８人の来客。結構忙しかった。


午後９時過ぎ閉店。
